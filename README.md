## Installation

```console
# clone the repo
$ git clone https://github.com/bugraozdmr/SilentEchoes.git

# change the working directory to SilentEchoes
$ cd SilentEchoes

# install the packages
pip install -r requirements.txt

```

## Usage

```console
$ python silentechoes.py 
```
